Version control records changes made to files to help the programmer recall specific versions later down the line.

Using the add command in git allows you to change the file in the next git commit.

Git commits can capture the state of a project at the time the command was written.

Git push commands can upload local repository content to a remote repository.

The pull command is the reverse of the push comman, it downloads remote repository content to a local repository

The git clone command will download a existing git repository to your pc.
